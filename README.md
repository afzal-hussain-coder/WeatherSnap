# WeatherSnap ☁️📱

WeatherSnap is a modern Android weather reporting application built using Kotlin, Jetpack Compose, MVVM, Clean Architecture, Room DB, Hilt, Retrofit, and CameraX.

## Features

* Search cities using Open-Meteo Geocoding API
* Weather details using Open-Meteo Forecast API
* Dynamic city suggestions
* Loading, empty, success, and error states
* Create weather reports
* Capture photos using CameraX
* Save reports locally using Room Database
* View saved reports with image preview
* Image compression support
* Modern Jetpack Compose UI
* Clean Architecture + MVVM

## Tech Stack

* Kotlin
* Jetpack Compose
* MVVM Architecture
* Clean Architecture
* Hilt Dependency Injection
* Retrofit
* Room Database
* CameraX
* Coil
